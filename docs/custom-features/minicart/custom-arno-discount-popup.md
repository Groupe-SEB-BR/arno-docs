# Custom Arno Discount Popup

 Este componente mostra um resumo de descontos aplicados no mini-cart e permite abrir um modal com detalhes.

## Usage

react/DiscountDetailsPopup.js

```jsx
import DiscountDetailsPopup from './components/DiscountDetailsPopup/index';

export default DiscountDetailsPopup;
```

store/interfaces.json

```json
  "custom-arno-discount-popup": {
    "component": "DiscountDetailsPopup"
  },
```
