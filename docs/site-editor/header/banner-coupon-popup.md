## Onde encontrar

Disponível no **Site Editor (VTEX)** em todas as páginas.

Caminho geral:

- Admin VTEX → CMS → Site Editor

## Visão geral

O componente **Coupon Banner Popup** permite exibir um pop-up com cupom e imagens responsivas em desktop e mobile.

## Campos do componente

### Imagem Desktop

Upload de imagem para exibição em resolução desktop.

### Imagem Mobile

Upload de imagem para exibição em resolução mobile.

### Tempo de exibição (minutos)

Define por quanto tempo o pop-up será exibido ao usuário (padrão: 1 minuto).

### Cupom Desktop

Código do cupom exibido para usuários em desktop (padrão: `VOLTA5`).

### Cupom Mobile

Código do cupom exibido para usuários em mobile (padrão: `VOLTA5`).

## Boas práticas

- Verifique se as imagens estão otimizadas para ambos os dispositivos
- Teste o tempo de exibição para melhor UX
- Revise o comportamento em diferentes tamanhos de tela antes de publicar

## Observações finais

- A gestão do conteúdo é feita diretamente pelo Site Editor
